## ✔️`Stash`
___
#### Armazenar uma alteração temporária.
```git
git stash
```
```git
git stash --include-untracked
```
___
```git
git stash apply
```
#### Aplicar uma alteração temporária
___
```git
git stash list
```
#### Listar alterações temporárias
___
```git
git stash clear
```
